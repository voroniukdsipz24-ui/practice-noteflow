Live-перегляд Markdown
Бачте відформатований результат поруч під час набору. Заголовки, блоки коду, таблиці та цитати миттєво відображаються з підсвіткою синтаксису для 70+ мов.
Панель форматування
Швидкі кнопки для жирного тексту, курсиву, заголовків, списків, цитат, посилань і коду. Кнопки контекстні — вони активуються, коли курсор знаходиться всередині відповідного синтаксису.
Безпечний доступ
Створюйте унікальне посилання для будь-якої нотатки. Додайте пароль, встановіть час дії або увімкніть режим одноразового перегляду — нотатка самознищується після першого відкриття.
Нотатки з самознищенням
Позначте нотатку як одноразову. Після відкриття отримувачем вона назавжди видаляється із сервера — без можливості відновлення. Підходить для конфіденційної інформації.
Захист паролем
Захистіть нотатку паролем перед поширенням. Отримувач повинен ввести правильний пароль, щоб відкрити вміст — навіть одноразові нотатки можуть бути захищеними.
Час дії нотаток
Встановіть автоматичне видалення через 1 годину, 24 години, 3 дні або 30 днів. Після завершення часу посилання перестає працювати, нічого очищати вручну не потрібно.
Миттєвий пошук
Повнотекстовий пошук по всіх нотатках у реальному часі. Знаходьте потрібне одразу під час введення — індексуються і заголовки, і вміст.
Експорт у Markdown
Завантажте будь-яку нотатку як файл .md одним кліком. Ваші нотатки залишаються переносимими та придатними для використання в майбутньому.
Темна та світла тема
Перемикайтеся між темною темою (Tokyo Night syntax) і світлою темою (GitHub syntax) — вибір зберігається у сесії.
